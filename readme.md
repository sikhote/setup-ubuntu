# setup-ubuntu
install as much as possible through cli

## install
```
cd ~/Downloads
wget https://github.com/sikhote/setup-ubuntu/archive/master.zip
unzip master.zip
rm -rf master.zip
cd setup-ubuntu-master
chmod +x beaver.sh
. beaver.sh
```
